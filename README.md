## 安装
1. 官网下载 Material Maker ([Itch.io](https://rodzilla.itch.io/material-maker) / [Steam](https://store.steampowered.com/app/4110830/Material_Maker/))
2. 打开设置 (Edit > Preferences)
3. 下载 [zh_CN.po](https://raw.githubusercontent.com/williamchange/material-maker-zh-cn/refs/heads/master/zh_CN.po) 文件
4. Install > 选择 "zh_CN.po" > `zh_CN, Chinese - China` > Apply
